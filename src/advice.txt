Quyidagi jadvalda backenddagi barcha endpointlarni (route'larni) to'liq ro'yxatini beraman. Barcha endpointlar /api prefiksi bilan boshlanadi (masalan, /api/auth/login). Men har bir modulni tekshirib, GET, POST, PATCH, DELETE so'rovlar uchun mavjud endpointlarni aniqladim.

Endpointlar quyidagi guruhlarga bo'lingan:

Public: Autentifikatsiyasiz ishlaydi.
Authenticated: Token (Bearer) talab qiladi.
Admin-only: Faqat admin roli bilan ishlaydi (authenticated + admin middleware).
Agar GET so'rovlar 401 qaytaryotgan bo'lsa, ehtimol admin panel token yubormayotgan yoki token noto'g'ri. Admin panelda barcha so'rovlar uchun Authorization: Bearer <token> header qo'shing.
